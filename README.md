 TCP Chat Application

 📌 Project Description

This project is a TCP-based chat application using Python sockets and Tkinter GUI.

 📁 Project Structure

 code → contains server and client programs
report → contains project report (DOCX)
 presentation → contains presentation slides (PPTX)
 screenshots → contains project images

 🚀 How to Run

 Server


python code/server.py


 Client


python code/client.py
